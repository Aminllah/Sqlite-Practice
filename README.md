# Todo API with SQLite

A simple **ASP.NET Core Web API** for managing Todo items using **SQLite**.  
Implements **CRUD operations**, **JWT authentication**, and **role-based authorization**.  

---

## Features
- Create, Read, Update, Delete Todo items
- JWT-based authentication
- Role-based authorization (Admin/User)
- Global exception handling middleware
- SQLite database support
- Structured API responses with DTOs

---

## Tech Stack
- **Backend:** ASP.NET Core 8.0 Web API
- **Database:** SQLite
- **ORM:** Entity Framework Core
- **Authentication:** JWT
- **Logging:** Serilog (optional)
- **Documentation:** Swagger
